# Swapnil Patil - Portfolio

A stunning, animated portfolio website that tells the journey of a Full Stack Developer from starting from scratch to leading high-impact teams.

## 🎯 Features

- **Animated Journey**: Scroll-triggered animations that tell a complete life story
- **7 Phases**: From Genesis to Persona, each phase represents a milestone
- **Interactive Elements**: Hover effects, progress bars, and smooth transitions
- **Responsive Design**: Works seamlessly on all devices
- **Modern Tech Stack**: Built with React, Framer Motion, and Tailwind CSS

## 🚀 Quick Start

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and visit:
```
http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

The optimized build will be in the `dist` folder.

## 🎨 Phases Overview

1. **Phase 0 - Genesis**: Terminal-style introduction with typewriter effect
2. **Phase 1 - Foundation**: Academic journey with interactive timeline
3. **Phase 2 - Pivot**: The academic gap story with parallax effects
4. **Phase 3 - Return**: Upskilling and internship at UptoSkills
5. **Phase 4 - Creation**: Major projects (Dairy-Tech, Yuzu, Movies App)
6. **Phase 5 - Zenith**: Current role as Team Manager
7. **Phase 6 - Persona**: Personal interests and contact information

## 🛠️ Tech Stack

- **React 18** - UI library

- **Framer Motion** - Animation library
- **Tailwind CSS** - Styling
